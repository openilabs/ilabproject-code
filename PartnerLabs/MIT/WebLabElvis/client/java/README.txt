ELVIS Client README

Compilation environment:

Sun JDK >= 1.4
Ant 1.6.5 (http://ant.apache.org/)


Instructions:

Ensure that the JDK and Ant are configured correctly and that javac,
javadoc, jar, and ant are on the executable path.

Go to the client/java directory

Type 'ant elvis' to build the production version of the ELVIS client.
The jar will be placed in the client/java/jar directory.

Type 'ant' for a list of other build options.

See test-graphical-applet.html for an example of invocation syntax and
parameters.
